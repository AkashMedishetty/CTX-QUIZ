# CTX Quiz - Product Overview

CTX Quiz is a real-time, synchronized quiz platform designed for live events (conferences, classrooms, game shows). Built by CTX.

## Core Capabilities
- 500+ concurrent users with sub-100ms timer synchronization
- Real-time WebSocket communication
- Multiple quiz modes: Regular, Elimination, Fastest-Finger-First (FFI)

## Client Components
1. **Admin Panel** - Quiz creation and management
2. **Controller Panel** - Live quiz control and moderation  
3. **Big Screen** - Projector/TV display for audiences
4. **Participant App** - Mobile-first player interface
5. **Tester Panel** - Diagnostics and load testing

## Key Features
- Comprehensive scoring (base points, speed bonus, streak bonus, partial credit)
- Session recovery with automatic reconnection
- Profanity filtering for nicknames
- Rate limiting and cheat prevention
- Offline answer queuing for participants
